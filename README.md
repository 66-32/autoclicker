# autoclicker

An autoclicker built with Python, pynput, and tkinter.

<br>

Features:

Can use left, right, middle mouse button and keyboard key

Can configure click interval (minutes, seconds, milliseconds)

F6 to start clicking, F7 to stop

<br><br>

Requirements

Python 3

<br>

pynput

pip install pynput

<br><br>

Usage

python autoclicker.py
